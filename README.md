`SiteScanner - Passive Web security Analizer`
A lightweight tool that performs passive website security checks by analyzing HTTP response headers.
Built using Node.js, Express, React, and Vite.

`Features`

- Detects missing security headers:

- HTTPS

- Content-Security-Policy

- Strict-Transport-Security

- X-Frame-Options

- X-Content-Type-Options

- Detects redirects to other domains

- Detects sensitive paths in robots.txt

- Simple and modern UI

`Tech stack`

- Frontend: React, Vite

- Backend: Node.js, Express

- Tools: Axios, Git

`App runs at:` http://localhost:5173

 `Usage`
`This tool performs passive scanning only.`
`Use only on websites you own or have permission to test.`
